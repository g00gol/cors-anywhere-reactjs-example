# Dependencies Used
React: [axios](https://www.npmjs.com/package/axios)  
Server: [CORS Anywhere](https://www.npmjs.com/package/cors-anywhere)  

# Installation
1. Clone this repository
2. Install server dependencies
  ```
  cd react  
  npm install
  ```
  * cd server
  * npm install
3. Start the app and server
  * ```cd react```
  * ```npm start```
  * ```cd server```
  * ```node server```
